Simple PHP MVC

Very basic PHP MVC, based on Sebastian Giroux Cassandra Cluster Admin.

View project onGitHub

If you wanted a very simple PHP MVC that is suitable for login-only sites, such as intranet apps, this might be the solution for you.

Based on Sebastien Giroux's Cassandra Cluster Admin, this is stripped of all those, leaving just a shell, controlled by an optional login.
